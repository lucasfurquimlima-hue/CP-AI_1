EstiloFácil
Um app que ultiliza IA para ajudar pequenos empreendedores a criar um guia de etilo completo para sua empresa, basta o cliente solicitar o que ele deseja para sua empresa que o app cria, definindo cores, tipografia e até o abordagem/tom de voz.
